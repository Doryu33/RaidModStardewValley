1) Ouvrir SMAPI 4.0.8 installer
2) Lancer install on Windows.bat
3) Suivre les instructions à l'écran
4) Prendre le dossier "Mods" et le coller dans: X:\...\Steam\steamapps\common\Stardew Valley\Mods
5) Modifier les options de lancement:
	- Clic droit sur Stardew Valley dans Steam
	- Proprieté
	- option de lancement, ajouter cette ligne: "X:\...\Steam\steamapps\common\Stardew ValleyStardewModdingAPI.exe" %command% 
	(NE PAS OUBLIER LES << " >>)
